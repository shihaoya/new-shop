# 功能需求规格说明书

## 1. 概述

### 1.1 功能背景

为平台管理员提供个人账号信息管理和邀请码生成能力。管理员可以通过个人中心修改个人信息，使用邀请码邀请店铺用户注册（店铺用户需审核后才能使用）。

### 1.2 本功能业务目标

1. 管理员可查看和修改个人账号信息（昵称）
2. 管理员可生成永久有效的邀请码用于邀请店铺用户注册
3. 管理员可重新生成邀请码，旧码立即失效，确保安全性

### 4.1 页面 1：管理员-个人中心

#### 4.1.1 页面概述

**页面形式：**独立页面

**页面职责：**为管理员提供个人信息查看/编辑和邀请码管理功能。管理员可在该页面修改昵称、生成/重新生成邀请码、复制邀请码。

#### 4.1.2 涉及字段

| 字段名称 | 字段类型 | 数据来源 | 获取时机 | 校验规则 | 默认值 | 业务含义 |
|---------|---------|---------|---------|---------|--------|--------|
| 昵称 | 文本 | 用户输入 | 实时输入 | 必填，1~30字符 | 无 | 管理员显示名称 |
| 当前邀请码 | 文本 | 数据库查询 | 页面加载 | 无 | 空 | 当前有效的邀请码（无则显示"暂无"） |
| 新邀请码 | 文本 | 系统生成 | 生成后 | 无 | 无 | 生成的新邀请码 |

#### 4.1.3 功能与按钮

**用户可操作功能：**

- **编辑个人信息**
  - 触发按钮：「编辑」按钮
  - 按钮级别：页面级
  - 按钮位置：个人信息区域右上角
  - 触发方式：点击按钮
  - 权限要求：仅管理员可操作
  - 加载状态：保存时显示 loading
  - 功能说明：点击后昵称输入框变为可编辑状态，显示「保存」和「取消」按钮

- **保存个人信息**
  - 触发按钮：「保存」按钮
  - 按钮级别：页面级
  - 按钮位置：编辑状态下的表单底部
  - 触发方式：点击按钮
  - 权限要求：仅管理员可操作
  - 加载状态：保存时显示 loading
  - 功能说明：点击后验证昵称格式（1~30字符），验证通过后保存到数据库

- **生成邀请码**
  - 触发按钮：「生成邀请码」按钮
  - 按钮级别：页面级
  - 按钮位置：邀请码管理区域
  - 触发方式：点击按钮（仅在当前无邀请码时显示）
  - 权限要求：仅管理员可操作
  - 加载状态：生成时显示 loading
  - 功能说明：点击后生成新的永久有效邀请码并展示

- **重新生成邀请码**
  - 触发按钮：「重新生成」按钮
  - 按钮级别：页面级
  - 按钮位置：邀请码管理区域（仅在已有邀请码时显示）
  - 触发方式：点击按钮
  - 权限要求：仅管理员可操作
  - 加载状态：生成时显示 loading
  - 功能说明：点击后弹出确认提示"重新生成后旧码将立即失效，是否继续？"，确认后生成新码，旧码立即失效

- **复制邀请码**
  - 触发按钮：「复制」按钮
  - 按钮级别：字段级
  - 按钮位置：邀请码输入框右侧
  - 触发方式：点击按钮
  - 权限要求：仅管理员可操作
  - 加载状态：无
  - 功能说明：点击后将当前邀请码复制到剪贴板，显示"已复制"提示

#### 4.1.4 业务规则

- **规则1：邀请码唯一性**
  - 规则描述：每次生成的邀请码必须全局唯一
  - 触发条件：生成邀请码时
  - 约束说明：系统通过 UUID 或时间戳+随机数确保邀请码不重复

- **规则2：邀请码失效规则**
  - 规则描述：重新生成邀请码后，旧码立即失效
  - 触发条件：点击"重新生成"并确认后
  - 约束说明：已在注册/审核中的店铺用户账号继续可用旧码完成注册流程

#### 4.1.5 交互逻辑

- **昵称编辑状态**：点击「编辑」后，昵称变为输入框可编辑状态，显示「保存」和「取消」按钮；点击「取消」或按 ESC 退出编辑状态
- **复制反馈**：复制成功后显示 2 秒 Toast 提示"已复制到剪贴板"
- **重新生成确认**：点击「重新生成」后弹出 Modal 确认对话框

#### 4.1.6 权限要求

> **参见第2.2节"页面访问权限"**，本页面的可访问角色、可操作角色、字段级权限均以第2章定义为准。
