# 功能需求规格说明书

## 1. 概述

### 1.1 功能背景

店铺用户需要对本店商品进行分类管理，以便对商品进行分组展示和筛选。商家自定义分类与系统内置分类（虚拟/实物）平级独立，商家可自行创建、编辑、删除本店商品分类。

### 1.2 本功能业务目标

1. 支持店铺用户创建本店商品分类
2. 支持店铺用户编辑、删除本店商品分类
3. 支持店铺用户查看本店所有商品分类列表
4. 支持分类的排序功能，便于调整展示顺序
5. 支持批量删除分类

```
[商品分类列表页] → 点击「新建分类」 → [新建分类弹窗]
                → 点击「编辑」 → [编辑分类弹窗]
                → 点击「删除」 → [确认弹窗]
                → 批量选择 → 点击「批量删除」 → [确认弹窗]
```

**流程说明：**
1. 用户进入商品分类列表页
2. 点击「新建分类」打开新建分类弹窗，填写完成后保存
3. 点击行内「编辑」打开编辑分类弹窗，修改后保存
4. 点击行内「删除」弹出确认框，确认后执行删除（系统检查商品关联）
5. 勾选多个分类后点击「批量删除」，弹出确认框确认后批量删除
