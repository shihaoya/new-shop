## 四、主题系统

### 4.1 可用主题

| 主题名称 | 主色 | 描述 |
|---------|------|------|
| `''` (空) | 科技蓝 | 默认主题 |
| `'fresh-green'` | 清新绿 | 绿色系主题 |
| `'romantic-purple'` | 浪漫紫 | 紫色系主题 |
| `'vibrant-pink'` | 活力粉 | 粉色系主题 |
| `'pure-white'` | 纯净白 | 深色主题 |

### 4.2 主题切换方法

```javascript
// 设置主题颜色
setTheme('fresh-green');  // 切换到清新绿
setTheme('');              // 恢复到默认科技蓝

// 切换暗色模式
toggleDarkMode();          // 切换明/暗模式

// 获取当前状态
console.log(isDark.value); // true = 暗色模式
```

### 4.3 CSS 变量应用

```css
/* 使用主题色变量 */
.my-element {
  background: var(--theme-primary);
  color: var(--color-on-primary);
  box-shadow: 0 8px 32px rgba(var(--theme-rgb-primary), 0.3);
}

/* 使用语义色 */
.success-text {
  color: var(--color-success);
}

/* 使用字体变量 */
.text-content {
  font-family: var(--font-body);
  font-size: 14px;
}
```

## 九、注意事项

1. **CDN 引用顺序不可变**：先 Tailwind，再 Vue，最后是 common.css/js
2. **主题状态自动保存**：刷新页面会恢复之前的主题设置
3. **所有颜色使用 CSS 变量**：确保主题切换正常工作
4. **图标使用 Material Symbols Outlined**：保持视觉一致性

## 十、常见问题

### Q: 如何修改侧边栏菜单？
A: 修改 `window.DEFAULT_NAV_ITEMS` 或在 `ThemeApp.init()` 时传入 `navItems` 参数。

### Q: 如何添加新页面？
A: 复制 `layout.html` 结构，在 `<main class="content-area">` 内添加页面内容。

### Q: 如何禁用主题切换？
A: 移除主题按钮和主题面板的 HTML 代码。

### Q: 如何添加自定义主题色？
A: 在 `common.css` 中添加新的 `[data-theme="xxx"]` 选择器。
